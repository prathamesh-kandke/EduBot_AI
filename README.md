# EduBot_AI

EduBot is an AI-powered smart education platform built with Flask and Gemini 2.5 Flash. It enables AI test generation, auto evaluation, adaptive quizzes, doubt solving, flashcards, analytics dashboards, and admin management for teachers, students, schools, and colleges.

---

## 🚀 Features

### 👨‍🏫 Teacher Panel
- Generate AI-based test papers instantly
- Create tests using manual input, custom prompts, PDF upload, or question bank
- View student performance analytics
- Export results in CSV format
- Print ready question papers

### 👨‍🎓 Student Panel
- Join tests using Test ID
- Practice mode for self-learning
- Adaptive quizzes based on performance
- AI doubt-solving assistant
- Smart flashcards generation
- Progress tracking dashboard

### 🔐 Admin Panel
- Secure admin login
- Manage teachers and students
- Monitor tests and submissions
- View complete platform analytics

### 🤖 AI Features
Powered by Gemini 2.5 Flash:
- Question generation
- Auto evaluation
- OCR text extraction
- Personalized study advice
- Flashcard generation
